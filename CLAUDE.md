# CLAUDE.md - Spark DE Certification Practice

## Project Overview
Practice repository for Databricks Certified Data Engineer Professional exam preparation.

## Key Commands

```bash
# Setup
python -m venv venv
source venv/bin/activate
pip install -r requirements.txt

# Run a problem file
python problems/01_dataframes/problems.py

# Start Jupyter
jupyter notebook
```

## Structure

- `datasets/` - Sample data (CSV, JSON)
- `problems/` - 10 topic folders with practice problems
- `solutions/` - Reference solutions
- `notebooks/` - Jupyter notebooks

## Topics
1. DataFrames basics
2. Transformations
